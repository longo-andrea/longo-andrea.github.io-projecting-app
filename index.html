Projecting

Keep your projects organized by having all information in one place.

Workflow

1. Add a Project

As first, you have to add a new project.
It's quite simple, only three info are required!
  • Insert project name
  • Insert a brief project description
  • Add deadlines' date

2. Add a Task

Then, add some tasks!
  • Insert task name
  • Insert a brief task description

From the project page you can also mark a task as working on!

3. Overview

Homepage allow you to fastly check the project progress.
There are some handful features:
  • You can mark tasks, deadlines and projects as completed
  • By tapping on task's name, you can see its description
